# 🚘car-arena
 
 Backend API for a CarData platform. It allows users to explore car models, compare specifications, and interact with car-related content through a mordern API-driven system 
 
 ## Features
 
 - Dynamic Database:Query cars by make, model, year, category, price, category, price, or specifications.
 - Filtering & Sorting
 - Recommendation engine
 - Favourites and saved list
 - Authentication & Security
 - Scalable Architechture

## Tech Stack 
 
 * FastAPI 
 * Next.js
 * PostgreSQL
 * Redis
 * Nginx / Docker

## Installation

 1. CLone the repository
 2. Setup virtual env. and install requirements
 3. Setup .env file for sensitive config
